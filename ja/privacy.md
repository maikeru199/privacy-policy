# プライバシーポリシー（英単語マスター2000）  

最終更新日：2025-11-24  
運営者名：マイケル  
お問い合わせ先：[syorinji5@gmail.com](mailto:syorinji5@gmail.com)  

本アプリ「英単語マスター2000」は、ユーザーのプライバシーを尊重し、個人情報の取り扱いについて以下のとおり定めます。  

## 1. 取得・利用する情報  
当アプリは、サービス提供と学習体験の向上のため、以下の情報を**端札内**に保存します。サーバーへ送信・保存は行いません。  
- **学習データ**：正誤履歴、学習・復習の進捗、学習目標など（Core Data で端札内保存）  
- **アプリ設定**：学習目標、表示・音声設定など（UserDefaults で端札内保存）  
- **購入状況**：アプリ内購買によるフルアクセス購入状態およびサブスクリプション登録状態（Keychain で端札内保存）  

※現時点で、位置情報・連絡先・写真・カメラ・マイク・通知などの端札機能の機能許可は要求しません。  

## 2. 外部送信・第三者提供  
- 当アプリでは広告配信および分析のために Google AdMob SDK 等の第三者サービスを使用します。これらのサービスは IP アドレスや広告 ID などのデバイス情報を取得する場合があります。詳細は各サービスのプライバシーポリシーをご參照ください。  
- ユーザーデータを第三者へ販売または提供することはありません。  

## 3. 決済情報  
アプリ内購買およびサブスクリプションは Apple の App Store を通じて処理されます。クレジットカード情報等の機密情報は当アプリでは保持しません。サブスクリプションは月額300円で、初回7日間は無料トライアルとして提供されます。購買は自動に更新され、解約されない限り更新日に費用が計算されます。  

## 4. サービス提供地域  
本アプリは EU 域内へのサービス提供を行っておらず、EU デジタルサービス法のトレーダー要件の対象外です。  

## 5. プライバシーポリシーの変更  
必要に当たって本プライバシーポリシーを更新することがあります。重大な変更がある場合はアプリ内またはウェブサイトで告知いたします。
